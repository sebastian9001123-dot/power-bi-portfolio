## 🎯 Qué hace este dashboard

Monitorea las toneladas gestionadas por diferentes asociaciones con filtros interactivos.

## 🔧 Funcionalidades

- Filtros por asociación y mes
- Gráfico de barras comparativo
- Distribución porcentual (gráfico circular)
- Total de toneladas acumulado

## 🛠️ Herramientas

Power BI | DAX | Power Query
