## 👋 Sobre este repositorio

Este repositorio contiene mis proyectos de **Power BI** como Analista de Datos. Cada dashboard demuestra habilidades en:

- ✅ Segmentación de datos (slicers)
- ✅ Visualizaciones interactivas
- ✅ Manejo de datos transaccionales
- ✅ Diseño de dashboards empresariales

---

## 📁 Proyectos

### 1. Reporte de Toneladas por Asociación 2025

**Descripción:** Dashboard interactivo para monitorear volumen de toneladas gestionadas por diferentes asociaciones.

**Funcionalidades:**
- Filtros interactivos por asociación y mes
- Gráfico de barras comparativo
- Distribución porcentual (pie chart)
- Tarjeta de total acumulado

---

### 2. Reporte Detallado de Toneladas y Valor por SAS

**Descripción:** Dashboard transaccional para analizar toneladas y valor monetario por mes, asociación y empresa gestora.

**Funcionalidades:**
- Tabla dinámica con totales
- Filtros por mes y asociación
- Cálculo de valor total
- Total toneladas
